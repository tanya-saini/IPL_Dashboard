# IPL Dashboard

## 📌 Project Overview  
This project presents an **IPL (Indian Premier League) Dashboard** built to analyze **season-wise and player-wise performance** of teams, batters, and bowlers. The dashboard provides interactive visualizations that help uncover patterns, trends, and key insights from IPL match data across multiple seasons.

The goal of this project is to **transform raw cricket data into meaningful insights** that support performance evaluation, strategy planning, and decision-making.

---

## 🎯 Objective  

The primary objectives of this dashboard are:

- Analyze **season-wise team performance trends**
- Evaluate **player-wise batting and bowling performance**
- Identify **top-performing batters and bowlers**
- Understand **match-winning patterns and consistency**
- Visualize performance metrics such as:
  - Runs scored  
  - Wickets taken  
  - Strike rate  
  - Economy rate  
  - Matches won  
- Build an **interactive dashboard** that allows users to explore IPL data dynamically.

---

## 🛠️ Tools & Technologies Used  

- **Power BI** — Dashboard creation and visualization  
- **Excel / CSV Dataset** — Data source  
- **Data Cleaning & Transformation** — Handling missing values and formatting  
- **DAX (Data Analysis Expressions)** — Creating calculated measures and KPIs
